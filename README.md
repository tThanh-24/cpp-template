# Competitive Programming Template

## Purpose

This repository contains my personal C++ template and reusable snippets for competitive programming.

The goal is to build a clean, minimal, and fully understood code base that I can rely on during contests.

## Philosophy

- Only include code that I fully understand.
- Clarity over excessive macros.
- Keep the template minimal.
- Improve it gradually after contests and lessons.

## Structure

- template.cpp — main contest template
- snippets/ — reusable data structures and algorithms (unavailable)
- stress_test/ — tools for testing solutions (unavailable)

## Compiler

g++ -std=gnu++14 -O2 -Wall -Wextra

## Notes

This template will evolve as I gain experience.
I will refactor it regularly to remove unnecessary parts and improve clarity.
